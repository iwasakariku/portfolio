# design.md — 陸駆 Portfolio Design System

## コンセプト

**「陸駆から、深海へ。」**

- 名前の「陸駆（りく）」＝陸と、海の対比を主題にする
- スクロール＝潜水。下に潜るほど背景色が濃くなり、人物の深層（技術・思想）に到達する
- セクションを実在の海洋区分（有光層／中深層／漸深層／深海底）に対応させ、水深をUIとして可視化する

## レイアウト

- 1カラム縦一本。各セクション `min-height: 100vh`、padding `14vh 12vw 14vh 10vw`
- コンテンツ幅 `max-width: 760px`、左寄せ
- 右端 84px はダイブメーター専用領域として常時確保

## シグネチャ要素：ダイブメーター

このサイトを記憶させる唯一の装飾。**ここ以外は静かに保つ。**

- 画面右端に固定。縦の基準線＋31本の目盛（5本ごとにmajor）
- スクロール率を水深 0〜3,000m に変換してモノスペースで数値表示
- 現在位置マーカー（発光ドット）が縦ラインに沿って移動

## モーション

| 効果 | 仕様 | 範囲 |
|------|------|------|
| 背景色補間 | scroll連動・rAFスロットリング | 全域 |
| 太陽光レイ | radial-gradient、スクロールで急速フェードアウト | 表層のみ |
| マリンスノー | Canvas粒子（最大90個）、−500m付近からフェードイン | 中層以深 |
| リビール | IntersectionObserver、translateY(28px)→0 / .9s | 各セクション |
| DIVE誘導 | 縦ライン内をドットが落下するループ | ヒーローのみ |

`prefers-reduced-motion: reduce` 時はリビール・粒子・誘導アニメをすべて無効化。

## コピーの方針

- 「潜水」のメタファーで統一。読み進める行為＝潜る行為として一貫させる
- ヒーロー：「名は陸駆。けれど仕事の本質は、いつも水面の下にある。」
- 最深部の締め：「ここまで潜ってくれた人とは、きっといい仕事ができる。」
- ラベル・計器類は英大文字Mono（DEPTH / DIVE / SURFACE等）、本文は日本語

## 品質基準

- レスポンシブ：720px以下でメーター幅を56pxに縮小、padding調整
- キーボードフォーカス可視（`outline: 2px solid var(--ink)`）
- スクロール処理は `requestAnimationFrame` + passive listener

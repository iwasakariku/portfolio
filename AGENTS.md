# AGENTS.md — 陸駆 Portfolio

## 潜水記録：プロジェクトの概要

このリポジトリは、エンジニア「陸駆（Riku）」のポートフォリオサイトを管理しています。
「陸駆」という名前が持つ「陸」のイメージと、そこから深く潜っていく「海」の対比をコアコンセプトに据えています。

単なるスキルの羅列ではなく、**「深く潜るほど、その人の本質（思想や技術の深層）に触れる」**という体験を、スクロールと連動した深度変化（0m 〜 -3,000m）で表現しています。

### 技術的なこだわり

- **Web標準への回帰**: ライブラリやフレームワークに依存せず、Vanilla HTML/CSS/JS のみで構築。静かな海の空気感を損なわないよう、軽量かつ高速な動作を重視しています。
- **計器としてのUI**: ダイブメーターや深度タグなど、情報を「計器」として提示。
- **Sky Overlay**: ヒーローエリアで「さらに上」へ向かうことで現れる隠しセクション。仕事の原則（Principles）を空の光の中で示します。

## 開発のガイドライン

### ローカルでの確認

ビルドステップはありません。
`index.html` をブラウザで開くか、`npx serve .` などの静的サーバーでホストしてください。

### デザインの整合性（`docs/design.md` を参照）

このプロジェクトにおいて、デザインは単なる装飾ではなく「体験」そのものです。

- **深度カラー**: スクロール位置に応じた背景色の補間（RGB線形補間）は、`js/script.js` 内の `stops` 配列で厳密に管理しています。
- **静けさの維持**: ダイブメーター以外の過剰な装飾は避けてください。

### パフォーマンスとアクセシビリティ

- **rAFスロットリング**: スクロール連動のアニメーションは、必ず `requestAnimationFrame` を経由させ、メインスレッドに負荷をかけないようにします。
- **Reduced Motion**: `prefers-reduced-motion` 設定時には、マリンスノーやリビールアニメーションを無効化し、酔いを防ぐ配慮を行っています。

## 主要なファイル構成

- `index.html`: サイトの構造。深度データ（`data-depth`）を各セクションが保持。
- `js/script.js`: 潜水体験の心臓部。スクロール計算、背景色補間、Canvas制御。
- `style/styles.css`: CSS Variables を活用したデザインシステム。
- `docs/design.md`: 思想からタイポグラフィまでを定義した設計書。
- `history.html`: これまでの歩みを記した別ページ。
